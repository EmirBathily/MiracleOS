## Que es esto?
Miracle Boot Screen es una aplicación que se ejecuta al iniciar el sistema otorga una animación de carga mientras se inicia el  sistema en segundo plano

La aplicacion esta basada en plymouth (Usa su codigo fuente)

## Instalación
Esta versión de plymouth esta preinstalada en miracle os pero si se puede instalar en otras distribuciones pero se debera compilar por el usuario y el mismo debe instalarlo
