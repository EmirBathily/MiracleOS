## Que es "Ajustes"?
Miracle Ajustes, es la app de ajustes de miracle OS, que esta basada en GNOME settings, es el encargado de habilitar la opcion de personalizar y cambiar las funcionalidades del sistema.

## Instalación
Esta aplicación viene preinstalada en Miracle os pero puedes buscar información en internet sobre como compilar
e instalar control center o settings solo que en ves de clonar desde el gitlab de gnome clonar desde el github de miracle(Esta pagina)
